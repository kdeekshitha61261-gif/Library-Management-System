Library Management System

Project Description
This is a simple Library Management System web application developed using HTML, CSS, and JavaScript. The system is designed to manage basic library operations such as searching books, issuing books, returning books, managing membership, user management, and generating reports.

The application follows a structured flow where users can log in and access different modules based on their role.

---

Features

Search Books
- Allows users to search books by name
- Displays matching results with author information

Issue Book
- User can enter book name
- Author name is automatically populated and non-editable
- Issue date is selected by user
- Return date is automatically set to 15 days and can be modified within allowed limit

Return Book
- Allows users to return issued books
- Includes serial number validation
- Fine status can be marked as paid or not paid

Maintenance Module
- Admin can add new books with author details
- Updates book list dynamically

User Management
- Supports new and existing users
- Requires user name input

Membership Module
- Supports 6 months, 1 year, and 2 years membership options
- Default selection is 6 months

Reports Module
- Displays all system activities such as book issue, return, and updates

Chart Module
- Shows system workflow:
Login → Search → Issue → Return → Reports
